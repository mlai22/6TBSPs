## Getting Started

The following steps should help you set up your environment:

### Clone Repo

```sh
git clone https://github.com/yge15/6TBSPs.git
```

### Install Requirements

Our code is tested using Python 3.8.
```sh
pip3 install -r requirements.txt
```
